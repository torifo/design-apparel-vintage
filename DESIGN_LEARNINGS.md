# FRAY Design Learnings

## 目的

vintageペルソナは、サステナブルな思想に共感しつつも、説教的で地味なサイトには惹かれにくい。FRAYでは、古さを欠点ではなく美しさとして提示し、読む時間そのものをブランド体験にした。

このブランドは架空ブランドであり、実在の店舗・商品ではない。

## 設計したこと

- 紙、活版印刷、セピア写真、長めの本文で、デジタルECではなく古い印刷物の質感を作った。
- ナビは固定せず、ページをめくる感覚を優先した。
- 商品には `入荷` `一点物` `修繕済` などのラベルを付け、在庫ではなく出会いとして見せた。
- プロセスセクションで素材、製造、配慮を明示し、思想を感覚だけで終わらせないようにした。

## CSS

主要なCSS判断:

```css
:root {
  --bg: #F2EBE0;
  --fg: #2C2018;
  --accent: #8B5C3E;
  --font-display: 'Playfair Display', 'Noto Serif JP', serif;
  --font-body: 'DM Serif Text', 'Noto Serif JP', serif;
}

body::before {
  opacity: .045;
  mix-blend-mode: multiply;
  background-image: url("data:image/svg+xml,...");
}

.story-copy {
  columns: 2 280px;
}
```

全体をセリフ体で統一し、紙のノイズを薄く重ねた。ノイズは主張しすぎると読みづらくなるため、低いopacityにしている。

## アニメーション

- ローダーは使わず、直接ページが現れる。
- revealは移動量を8pxに抑え、現代的なスライド感を避けた。
- 商品ホバーはセピアを少し戻し、古いものに触れた時の発見を表現した。

## フォント

- Display: `Playfair Display`
- Body: `DM Serif Text`

見出しも本文もセリフ体にし、活版印刷や古書の連想を強めた。本文の情報量が多いため、行間を広めにして読める速度を落としている。

## ボタン

FRAYでは強いボタンを前面に出さない。ニュースレターやカテゴリも線と文字を中心にし、購入よりも納得して選ぶ体験を優先した。

## UI/UX

- ヒーローは大きな写真とブランド名で、静かな信頼感を作る。
- ストーリーは2段組で、思想をじっくり読ませる。
- プロセスは3ステップで、素材・製造・配慮を短く整理する。

## レイアウト

不規則グリッド、傾いた写真、スタンプ風ラベルで古着屋の棚を表現した。ARCHのような攻撃的な崩しではなく、偶然見つける感覚を作るための揺らぎとして使っている。
