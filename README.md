# FRAY — apparel-vintage

> **"Worn with intention"**

A design study exploring a fictional youth apparel brand tailored to the **vintage persona** — eco-conscious, thrift-culture, slow-fashion aesthetic, targeting Japanese creatives aged 22–27.

FRAY is a fictional brand created for this design study. It is not a real brand, store, or product.

## Overview

| | |
|---|---|
| **Brand** | FRAY |
| **Persona** | vintage |
| **Live Site** | [torifo.github.io/apparel-vintage](https://torifo.github.io/apparel-vintage/) |
| **Custom Domain** | `design.apparel-vintage.riumu.net` *(planned — not yet active)* |

## Design Concept

- **Color**: Oatmeal `#F2EBE0` × terracotta `#8B5C3E` × deep walnut `#3D3028`
- **Typography**: Playfair Display (display) × DM Serif Text (body)
- **Aesthetic**: Letterpress poster × vintage shop shelf
- **UX**: Paper texture overlay, intentionally irregular grid, sepia image filter, text-heavy philosophy section

## Tech Stack

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN
- No framework, no build step — GitHub Pages ready

## Part of

This repository is one of four design studies under the **apparel persona series**:
